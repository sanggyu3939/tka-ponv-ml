# Prediction of postoperative nausea and vomiting within 24 hours after total knee arthroplasty using interpretable machine learning

This repository accompanies the manuscript:

> **Prediction of postoperative nausea and vomiting within 24 hours after total knee arthroplasty using interpretable machine learning**

This repository is intended to provide the model specification and reproducibility materials required for independent evaluation of the final random forest model using the Step 3 feature set.

## Repository status

The reproducibility package is currently being prepared as part of the manuscript revision. The final release will include:

- the complete Step 3 predictor list and coding scheme;
- data preprocessing procedures;
- final model hyperparameters and random seeds;
- model-development and evaluation scripts;
- the serialized final random forest model;
- an inference script for external validation;
- an example input template; and
- software environment and package versions.

## Data availability

Patient-level clinical data are not included because they are protected by institutional privacy and data-protection requirements. The public reproducibility package will contain no directly or indirectly identifiable patient information.

## Intended use

The model and accompanying code are provided for research and independent validation. They are not intended for direct clinical decision-making without further external and prospective validation.
